# LAVERRAP - Sistema de Gestión para Lavadero de Autos
## Descripción del Proyecto
LAVERRAP es una aplicación web integral diseñada para la gestión moderna de un lavadero de autos. Combina funcionalidad completa con una interfaz intuitiva para optimizar las operaciones diarias del negocio.

### Arquitectura del Sistema
#### Frontend (React + TypeScript)
Framework: React 19 con TypeScript para robustez y escalabilidad
Build Tool: Vite para desarrollo rápido y optimización
Styling: Tailwind CSS 4 con sistema de diseño moderno
UI Components: Shadcn/ui para componentes consistentes y accesibles
Estado: TanStack Query para manejo eficiente de datos del servidor
Formularios: React Hook Form + Valibot para validación robusta
Routing: React Router 7 para navegación fluida
#### Backend (Node.js + Express)
Servidor: Express.js para API REST
Base de Datos: Supabase (PostgreSQL) con autenticación integrada
Integración: Twilio para notificaciones WhatsApp automáticas
CORS: Configurado para comunicación segura frontend-backend
